# EV-Population-Prediction

## Project Overview

This project focuses on preprocessing the Electric Vehicle (EV) Population dataset to prepare it for machine learning. It includes Exploratory Data Analysis (EDA), data cleaning, encoding categorical features, and scaling numerical features.

---

## Objectives

* Perform Exploratory Data Analysis (EDA).
* Clean and preprocess the dataset.
* Handle missing values and duplicate records.
* Encode categorical variables.
* Scale numerical features for consistent model input.

----

## 📂 Dataset

- **Dataset:** Electric Vehicle Population Data
- **Source:** Data.gov
- **Type:** Structured tabular dataset
- **Target Variable:** Electric Vehicle Type
- **Link:** https://catalog.data.gov/dataset/electric-vehicle-population-data?from_hint=eyJzb3J0IjoicG9wdWxhcml0eSJ9

----

## Features

* Exploratory Data Analysis (EDA)
* Data Cleaning
* Missing Value Handling
* Duplicate Removal
* Categorical Feature Encoding
* Numerical Feature Scaling

----

## Preprocessing Pipeline

The following preprocessing steps were performed:

1. Imported the dataset.
2. Performed Exploratory Data Analysis (EDA).
3. Identified and handled missing values.
4. Removed duplicate records.
5. Checked data types and corrected them where necessary.
6. Encoded categorical features using appropriate encoding techniques.
7. Applied feature scaling to numerical features.

## Instructions to Run the Project

1. Open the project notebook in **Google Colab**.
2. Upload the **EV Population** dataset (`.csv`) to the Colab session.
3. Install the required libraries (if they are not already available).
4. Run all the notebook cells sequentially.
5. The notebook will:

   * Load the dataset
   * Perform Exploratory Data Analysis (EDA)
   * Clean the data
   * Encode categorical features
   * Apply feature scaling
